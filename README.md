# 🎓 Controle Escolar

Sistema web desenvolvido para auxiliar no gerenciamento de equipamentos e reservas de uma instituição de ensino. O projeto permite controlar os equipamentos disponíveis, registrar manutenções e realizar reservas de forma organizada.

## 🚀 Funcionalidades

- Autenticação de usuários.
- Controle de acesso por perfil (Administrador, Professor e Técnico).
- Cadastro, edição e exclusão de equipamentos.
- Controle de disponibilidade dos equipamentos.
- Reserva de equipamentos por professores.
- Registro e acompanhamento de manutenções.
- Dashboard com informações gerais do sistema.
- Filtro de equipamentos por laboratório.
- Interface responsiva e intuitiva.

## 👥 Níveis de acesso

### Administrador
- Gerenciar usuários.
- Gerenciar equipamentos.
- Visualizar todas as reservas.
- Gerenciar manutenções.

### Professor
- Consultar equipamentos disponíveis.
- Realizar reservas.
- Visualizar suas reservas.

### Técnico
- Registrar manutenções.
- Atualizar o status dos equipamentos.
- Consultar equipamentos cadastrados.

## 🛠️ Tecnologias utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP
- phpMyAdmin

## 🗄️ Banco de Dados

O sistema utiliza o MySQL para armazenar informações como:

- Usuários
- Equipamentos
- Reservas
- Manutenções
- Tipos de equipamentos

## 💻 Objetivo

Este projeto foi desenvolvido como atividade prática do Curso Técnico em Desenvolvimento de Sistemas, com o objetivo de aplicar conhecimentos em desenvolvimento web, banco de dados, autenticação de usuários e controle de permissões.

## 📚 Principais conhecimentos aplicados

- Desenvolvimento Full Stack com PHP.
- CRUD completo.
- Sessões e autenticação.
- Controle de permissões.
- Integração com banco de dados MySQL.
- Organização de código.
- Responsividade.
- Manipulação do DOM com JavaScript.

## ▶️ Como executar

1. Clone este repositório.
2. Copie a pasta para o diretório `htdocs` do XAMPP.
3. Inicie o Apache e o MySQL.
4. Importe o banco de dados pelo phpMyAdmin.
5. Configure a conexão com o banco de dados.
6. Acesse:

```
http://localhost/controle-escolar
```
