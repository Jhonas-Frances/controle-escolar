# Controle Escolar

Sistema web para gerenciamento de equipamentos escolares,
reservas e controle de manutenções.

## Tecnologias

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Funcionalidades

- Login com níveis de acesso
- Reserva de equipamentos
- Controle de manutenção
- Histórico de manutenções
- Controle por laboratório
- Sessões de usuário

## Tipos de Usuário

### Administrador
- Controle total

### Professor
- Reservas
- Visualização de manutenção

### Técnico
- Equipamentos
- Manutenções
- Histórico técnico

## Estrutura

/assets
/config
/pages

## Como executar

1. Instalar XAMPP
2. Colocar o projeto em htdocs
3. Importar banco MySQL
4. Iniciar Apache e MySQL
5. Abrir localhost/controle_escolar
